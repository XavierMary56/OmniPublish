# OmniPublish V2.0 — 用户操作手册

## 一、登录

1. 浏览器打开 `http://服务器地址:9527`
2. 输入管理员分配的**用户名**和**密码**
3. 点击「登录」进入工作台

> 忘记密码请联系管理员重置。

---

## 二、工作台（首页）

登录后默认进入工作台，一眼看到今日工作全貌：

- **统计卡片**：今日发帖数 / 已完成 / 进行中 / 失败 / 覆盖平台
- **最近任务**：最近的发帖任务列表，可直接点击进入
- **待我处理**：需要你确认文案、选封面、处理失败的任务

---

## 三、发帖流水线（核心功能）

### 如何新建发帖任务

1. 点击顶部「＋ 新建发帖任务」或侧边栏「发帖流水线」
2. 按照 6 个步骤依次操作

### Step 1：素材 & 平台选择

1. **选择素材文件夹**：拖入文件夹或手动输入路径
   - 系统自动识别图片、视频、文案文件数量
2. **选择发布平台**：
   - 平台按部组分组显示
   - 支持搜索平台名称
   - 点击「全选本组」可快速选择整组
   - 可同时选多个平台（一次发多个）
3. 点击「下一步」

### Step 2：文案生成

1. 填写素材信息：主角、事件、生活照描述、视频内容
2. 选择文风（反转打脸风 / 搞笑玩梗风 / 麻辣吐槽风 等）
3. 选择分类（根据你选的平台自动加载分类库）
4. 点击「AI 生成文案」
5. 查看生成结果：
   - 可直接在右侧编辑修改标题、关键词、正文
   - 不满意点「重新生成」
6. 满意后点「确认文案，下一步」

### Step 3：图片重命名

1. 系统自动用标题作为重命名前缀
2. 可修改前缀、起始编号、分隔符
3. 预览区实时显示重命名效果
4. 确认后点「确认重命名，下一步」

### Step 4：封面制作

1. 系统自动生成 3 个候选封面（AI 智能裁剪）
2. 点击选中一个封面
3. 不满意可「重新生成」或「手动选图」
4. 确认后点「使用选中封面，下一步」

### Step 5：水印处理

1. 系统展示各平台的水印方案（自动从业务线配置读取）
2. 检查各平台水印是否正确
3. 点击「确认水印方案，开始处理」
4. 等待处理完成（各平台显示独立进度）
5. 全部完成后自动进入下一步

### Step 6：上传 & 发布

1. 各平台并行上传视频，显示上传进度
2. 上传完成后等待平台切片
3. 切片完成的平台显示「存草稿」按钮
4. 可单个平台「存草稿」，或「一键发布所有已就绪平台」
5. 失败的平台可点「重试」

## 四、任务看板

### 查看所有任务

- 侧边栏点击「任务看板」
- 使用筛选按钮：全部 / 进行中 / 待确认 / 切片中 / 已完成 / 失败
- 搜索框支持按任务 ID、标题、平台名搜索

### 查看任务详情

- 点击任务行左侧箭头展开，查看各平台的独立分发状态
- 点击「详情」进入任务全貌

### 任务状态说明

| 状态 | 颜色 | 含义 |
|------|------|------|
| 进行中 | 蓝色 | 流水线自动执行中 |
| 待确认 | 橙色 | 需要你确认文案或选封面 |
| 切片中 | 橙色 | 视频已上传，等待平台切片 |
| 已完成 | 绿色 | 全部平台发布成功 |
| 失败 | 红色 | 发布失败，可重试 |

## 五、工具箱

工具箱中的工具可独立使用（不在流水线内），用于特殊需求：

| 工具 | 说明 |
|------|------|
| 遮盖四角水印 | 模糊遮盖视频四角水印区域 |
| 裁掉四角水印 | 裁剪掉四角水印区域 |
| 虚化填充 | 模糊背景填充为标准尺寸 |
| 片头片尾处理 | 裁掉或添加片头片尾 |
| 视频合成 | 多段视频排序合并 |
| 视频压缩 | 批量压缩到指定大小 |
| 独立文案生成 | 不走流水线单独生成文案 |

## 六、业务线管理（管理员）

### 新增业务线

1. 侧边栏 → 业务线管理 → 「＋ 新增业务线」
2. 填写：
   - 名称、部门
   - 分类库（回车逐个添加，或「批量导入」）
   - 图片水印（拖拽上传 PNG，设置位置和宽度）
   - 视频水印（拖拽上传 MOV/PNG，选择模式）
   - API 入口和项目代码
   - 排版模板
3. 点击「保存」

### 编辑业务线

点击业务线卡片上的「编辑」按钮。

### 水印文件管理

- 上传后显示缩略图预览
- 点击「替换」可更新水印文件
- 水印文件存储在服务端，无需每次手动选择

---

## 七、账号管理（管理员）

管理各平台的后台登录凭据（用于 API 发帖）：

- 查看各平台登录状态（已登录 / 已过期）
- 过期的账号点击「重新登录」
- 用户名和 URL 默认脱敏显示

---

## 八、数据统计

- 支持时间维度：今日 / 本周 / 本月 / 自定义
- 各平台发帖统计表
- 各编辑发帖排名
- 流水线各步骤耗时分析（找到瓶颈优化）

| 快捷键 | 功能 |
|--------|------|

## 十、常见问题

**Q: 文案生成失败怎么办？**
A: 系统会自动重试 3 次。如仍失败，检查网络连接，或联系管理员确认 API Key。

**Q: 水印效果不对怎么办？**
A: 在业务线管理中重新上传正确的水印文件，或修改水印位置/缩放参数。

**Q: 发布失败重试还是失败？**
A: 检查账号管理中对应平台的登录状态。如已过期，先重新登录再重试。

**Q: 为什么分类字段没有出现？**
A: 说明你选择的所有平台都没有配置分类库。联系管理员在业务线管理中添加分类。
