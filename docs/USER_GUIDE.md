## 一、登录与界面概览

### 登录

1. 浏览器打开 `http://服务器地址:9527`（或管理员配置的域名）

## 三、发帖流水线（核心功能）

### 新建发帖任务

点击顶部「＋ 新建发帖任务」或侧边栏「发帖流水线」，进入 6 步向导。

> **提示**：每步均可点击「← 上一步」返回修改，已填写的数据不会丢失。

---

### Step 1：素材 & 平台选择

**目标**：告诉系统这次发帖的素材在哪、要发到哪些平台。

**操作步骤**：

1. **选择素材文件夹**
   - 方式 A（推荐）：在「本地路径」输入框填入素材文件夹的绝对路径，点击「使用本地路径」
   - 方式 B：点击「上传文件」拖入文件夹（适合远程访问场景）
   - 系统自动扫描识别：图片 / 视频 / 文案文件数量

2. **选择发布平台**
   - 平台按部组分组展示（如「1组」「2组」）
   - 支持搜索框实时筛选平台名
   - 「全选本组」快速勾选整组
   - 可同时选多个平台，一次任务多平台并行发布

3. 点击「下一步 →」

---

### Step 2：文案生成

**目标**：通过 AI 生成适合各平台的标题、关键词和正文。

**操作步骤**：

1. **填写素材信息**（左侧表单）

   | 字段 | 说明 | 是否必填 |
   |------|------|---------|
   | 主角 | 内容主角描述（如「22岁女生」） | ✅ 必填 |
   | 事件 | 核心事件（如「闺蜜婚礼上当众揭穿渣男」） | ✅ 必填 |
   | 生活照描述 | 图片内容概括 | 选填 |
   | 视频内容 | 视频主要内容 | 选填 |
   | 文风 | 反转打脸风 / 搞笑玩梗风 / 麻辣吐槽风 等 | 选填 |
   | 分类 | 根据所选平台自动加载分类库，可多选 | 选填 |
   | 标题字数 / 关键词数 / 正文字数 | 控制 AI 输出长度 | 可调整 |

2. 点击「🤖 AI 生成文案」（需 10-30 秒）

3. **查看并编辑生成结果**（右侧区域）
   - 可直接编辑标题、关键词、正文
   - 不满意点「🔄 重新生成」
   - 分类已自动取各平台分类库的并集，发布时每个平台只使用自己支持的分类

4. 满意后点「✅ 确认文案，下一步 →」

---

### Step 3：图片重命名

**目标**：按标题关键词统一规范图片文件名。

**操作步骤**：

1. 「重命名前缀」已从标题自动提取，可手动修改
2. 调整「起始编号」和「分隔符」（下划线 `_` / 短横线 `-`）
3. 预览区实时显示新旧文件名对比
4. 点击「✅ 确认重命名，下一步 →」

> 说明：重命名操作在服务端执行，只修改文件名，不移动或复制文件。

---

### Step 4：封面制作

**目标**：为发帖选择一张封面图（横版 / 竖版均支持）。

**操作步骤**：

1. **配置封面参数**

   | 参数 | 选项 |
   |------|------|
   | 拼接方式 | 三拼（默认）/ 单图 / 双拼 / 宽屏横版 / 竖版 |
   | 封面尺寸 | 1300×640（标准） / 800×450 / 900×1200（竖版） |
   | 头顶留白 | 0~50%，为标题留出空间 |

2. 点击「🖼️ 生成封面候选」（约 5-15 秒）

3. 系统生成 3 个候选封面（AI 智能识别人脸并裁剪），点击选中满意的

4. **不满意时的选项**：
   - 「🔄 重新生成」：随机换一批候选
   - 「📁 手动选图」：从素材图片中手动选 1-3 张拼接

5. 选好后点「✅ 使用选中封面，下一步 →」

> **注意**：确认封面后如需返回修改，点「← 上一步」，封面候选会保留无需重新生成。

---

### Step 5：水印处理

**目标**：为各平台的图片和视频添加对应水印。

**操作步骤**：

#### 5.1 查看水印方案

页面展示所有平台的水印配置（从业务线管理中读取）：

```
海角社区          [图片+视频]
  haijiao_wm.png · 右下角 · 264px  |  haijiao.mov · 四角轮转 · 30%

小红书            [仅图片]
  xhs_wm.png · 左下角 · 200px
```

#### 5.2 单平台水印微调（可选）

如需调整某个平台的水印参数，点该平台右侧「✏️ 编辑」按钮展开编辑区：

| 参数 | 可选值 |
|------|-------|
| 图片水印位置 | 右下角 / 左下角 / 右上角 / 左上角 / 居中 |
| 图片水印宽度 | 50~800 px |
| 视频水印模式 | 四角轮转 / 固定位置 / 双水印对角 |
| 视频水印缩放 | 5~80 % |

修改后点「✓ 完成编辑」，参数仅对本次任务生效（不修改业务线配置）。

#### 5.3 确认并处理

1. 点击「✅ 确认水印方案，开始处理 →」
2. 各平台独立显示处理进度（并行处理，互不阻塞）
3. 等待所有平台完成（图片约 1 分钟 / 视频按时长计）

#### 5.4 无水印平台

橙色提示框列出未配置水印的平台，这些平台会跳过水印直接进入发布。

#### 5.5 完成确认

全部平台显示「✅ 完成」后，点「✅ 确认完成，进入上传 & 发布 →」。

> 若有平台处理失败，可选择「跳过失败，继续发布」。

---

### Step 6：上传 & 发布

**目标**：将带水印的内容上传并发布到各平台后台。

**操作步骤**：

1. 点击「🚀 一键发布所有已就绪平台」
2. 各平台并行上传，实时显示上传进度
3. 上传完成后平台开始视频切片（等待平台处理）
4. 切片完成自动触发发布

**各平台状态说明**：

| 状态 | 含义 |
|------|------|
| 排队等待 | 等待上传 |
| ⏳ 发布中 | 正在上传或切片 |
| ✅ 发布成功 | 已发布到平台 |
| ❌ 发布失败 | 发布失败，可点「🔄 重试」 |

**失败处理**：
- 点「🔄 重试」对单个平台重新发布
- 如多次失败，检查「账号管理」中该平台登录状态

## 四、任务看板

### 查看任务列表

侧边栏点击「任务看板」，可查看所有历史任务。

**筛选选项**：

| 筛选 | 含义 |
|------|------|
| 全部 | 所有任务 |
| 进行中 | 正在处理某一步 |
| 待确认 | 等待用户确认文案/封面 |
| 已完成 | 全部平台发布成功 |
| 失败 | 有平台发布失败 |

**搜索**：支持按任务编号、标题关键词搜索。

### 任务状态说明

| 状态 | 颜色 | 含义 |
|------|------|------|
| 进行中 | 蓝色 | 流水线自动执行中 |
| 待确认 | 橙色 | 需要操作员确认（文案/封面等） |
| 切片中 | 橙色 | 视频已上传，等待平台转码 |
| 已完成 | 绿色 | 全部平台发布成功 |
| 失败 | 红色 | 有平台发布失败，可重试 |

### 查看任务详情

- 点击任务行左侧「▶」展开，查看各平台独立状态
- 点击「详情」进入任务全貌（含操作日志）
- 失败的任务可在详情页直接重试单个平台

## 五、业务线管理（管理员）

业务线 = 平台账号的发布配置。每个业务线对应一个发布账号。

### 新增业务线

1. 侧边栏 → 业务线管理 → 「＋ 新增业务线」
2. 填写基本信息：

   | 字段 | 说明 |
   |------|------|
   | 名称 | 平台展示名，如「海角社区」 |
   | 部门 | 所属部组，如「1组」 |
   | API 入口 | 发布接口地址 |
   | 项目代码 | 平台标识代码 |

3. **分类库配置**：
   - 逐个输入分类后回车添加
   - 或点「批量导入」上传 CSV/TXT 文件（每行一个分类）

4. **水印配置**：

   - *图片水印*：拖拽上传 PNG，设置位置和宽度（px）
   - *视频水印*：拖拽上传 MOV/PNG，选择水印模式（四角轮转 / 固定 / 双对角）和缩放比例

5. **排版模板**（可选）：配置标题/正文的格式模板

6. 点「保存」

### 编辑业务线

点击业务线卡片上的「编辑」按钮，修改后保存。

### 水印文件管理

- 上传后显示缩略图预览（水印文件存储在服务端）
- 点「替换」更新水印文件
- 水印位置可选：右下角 / 左下角 / 右上角 / 左上角 / 居中

## 六、账号管理（管理员）

管理各平台后台的登录凭据，用于 API 自动发帖。

**账号状态**：

| 状态 | 颜色 | 含义 |
|------|------|------|
| 已登录 | 绿色 | Token 有效，可正常发布 |
| 已过期 | 红色 | Token 失效，需重新登录 |

**操作**：
- 「重新登录」：重新录入账号密码，刷新 Token
- 凭据脱敏显示，点击可切换显示/隐藏

## 七、数据统计

侧边栏 → 数据统计，支持多维度数据分析。

**时间维度**：今日 / 本周 / 本月 / 自定义区间

**统计内容**：

| 图表 | 说明 |
|------|------|
| 发帖趋势 | 每日发帖数量折线图 |
| 平台分布 | 各平台发帖数占比 |

| 流水线耗时 | 各步骤平均耗时（找出瓶颈） |

## 八、常见问题

### 文案生成

**Q: AI 生成的文案质量不好怎么办？**
A: 点「🔄 重新生成」可多次生成。也可直接在右侧编辑框手动修改后确认。

**Q: 文案生成一直转圈不出结果？**
A: 等待最多 60 秒。超时后请联系管理员确认 API Key 是否有效，或检查 VPS 的外网访问情况。

**Q: 分类选项为空？**
A: 说明所选平台均未配置分类库。联系管理员在业务线管理中添加分类。

### 封面制作

**Q: 生成的封面人脸歪了/裁剪不对？**
A: 点「🔄 重新生成」或用「📁 手动选图」自选图片和拼接顺序。

**Q: 返回上一步后封面消失了？**
A: 封面数据已保存，点「← 上一步」后候选会自动恢复。如已刷新页面，任务数据从服务端重新加载，候选仍在。

### 水印处理

**Q: 某平台水印效果不对（位置/大小）？**
A: 在水印处理步骤，点该平台卡片右侧「✏️ 编辑」，调整位置和宽度，本次任务立即生效。如需永久修改，到「业务线管理」中更新该平台的水印配置。

**Q: 视频水印处理很慢？**
A: 视频水印是逐帧处理，时长较长的视频需要几分钟。等待进度条完成即可，处理期间可切换到任务看板做其他任务。

**Q: 某平台提示「未配置水印，已跳过」？**
A: 该平台的业务线配置中未上传水印文件，联系管理员上传后重新处理，或直接跳过继续发布。

### 发布

**Q: 发布失败原因是什么？**
A: 点「❌ 发布失败」下方的「⚠️ 错误信息（点击展开）」查看详细错误。常见原因：
- Token 过期 → 账号管理中重新登录
- 视频格式不支持 → 检查视频编码
- 平台限流 → 等待几分钟后重试

**Q: 一次任务可以发多少个平台？**
A: 理论上不限，各平台并行处理。实际建议 10 个以内，避免占用过多服务器资源。

**Q: 发布后如何确认平台收到了内容？**
A: 看该平台状态显示「✅ 发布成功」。发布成功不代表内容立即公开，部分平台有审核流程。

### 账号与权限

**Q: 普通用户和管理员有什么区别？**
A: 管理员可访问业务线管理、账号管理；普通用户只能操作发帖流水线和查看自己的任务。

**Q: 密码如何修改？**
A: 当前版本需联系管理员在后台重置密码。
