## 核心开发任务总结

| 优先级 | 任务 | 说明 |
|--------|------|------|
| P0 | 用户认证系统 | 编辑独立账号登录 + 角色权限（编辑/组长/管理员） |
| P0 | 流水线引擎 | 6 步状态机 + 自动推进 + 人工确认阻塞 |
| P0 | 任务持久化 | 数据库存储任务状态，支持断点续传 |
| P0 | 多平台并行处理 | Step 5 水印并行 + Step 6 上传并行 |
| P1 | 前端重构 | 从静态 HTML 改为 SPA 框架（推荐 Vue/React） |
| P1 | 后端 API 重构 | 从通用 /api/run 改为 REST API |
| P1 | 业务线配置管理 | 31 个平台的水印/API 配置 CRUD |
| P2 | 任务看板 | 任务列表 + 多平台子任务展开 |
| P2 | 数据统计 | 按平台/编辑/时间维度统计 |
| P2 | WebSocket 实时推送 | 替代 SSE，推送流水线进度 |
